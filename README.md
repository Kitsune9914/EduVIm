# EduVim
Minha configuração do LazyVim

### Requisitos

- Ter o NeoVim instalado

# ⚠️  Atenção

Se já existir uma configuração do wezterm lembre de fazer backup

```
# Nescessario
mv ~/.config/nvim{,.bak}

# Opcional mas recomendavel
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```
